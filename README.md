# 🎙️ XTTS v2 Continuous Narration Pipeline

## 🎯 The Vision
Modern AI voice synthesis often struggles with long-form content. This pipeline provides a production-ready solution for generating high-fidelity, continuous narration by leveraging XTTS v2.

## ✨ Key Features
* **Intelligent Text Chunking:** Maintains narrative flow across long documents.
* **Audio Optimization:** FFMPEG-driven processing for broadcast-quality output.
* **Production-Ready:** Modular architecture for easy maintenance.

## 🛠 Technical Insights
* **System Architecture:** Separated frontend (Gradio) and backend (`audio_utils.py`).
* **XTTS v2 Integration:** High-fidelity voice cloning.
* **Performance Pipeline:** Automated noise reduction and normalization.

## 🚀 Quick Start
1. `git clone <repository-url>`
2. `chmod +x setup.sh && ./setup.sh`
3. `python app.py`
